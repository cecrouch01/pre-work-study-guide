HTML

  • HTML is the structure of the content of a webpage
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • There are other elements like sections as well
  • Every HTML element is basically a box

CSS

  • CSS stands for Cascading Style Sheets
  • Programming language used for defining how webpage content is presented to users
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • Inline CSS is used to style elementals directly, they apply to specific HTML tags
  • Interal Style Sheets contain CSS rules in the head section, not the body.
  • External CSS Style sheet is the most used method because it keeps all CSS rules ina separate file, making design changes easier
  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside an element
  • Declarations contain two important compontents, property and value
  • Multiple declarations can be in a selector and are called declaration blocks

  • Every time you do something with a webpage, even as simple as clicking a button, java is a work
  • To declare a variable you use the var keyword. Then give the variable a name
  • The equal sign is used to assign the specific data that we want stored in our variable.
  • A String is contained within a quotation mark and is a list of the variables
  • Javascript is linear, the order in which you put things matters
  • Javascript has conditional statements and loops to cycle through code
  • If statements are first checked to be truthy and if they are then the statement in the brackets will happen
  • An array is a single variable that used to hold a group of data, which is usually related
  • Arrays are useful for storing multiple itesm with different data. However, not all arrays have different data
  • To work with individual items inside an array, we need the location of the data item in the array. This location has a unique id called an index
  • Many programming languages are zero indexed, where zero is the first number in a sequence
  • Loops help with large arrays
  • A FOR loop uses a predictible patteren of indices to execute a single code of block repeatedly. A conditional if statement can interrupt the control flow so a block of code only executes a specific amount of times
  • A FOR loop contains a starting point for our loop, a condition, and a statement allowing arrays to iterate over each item
