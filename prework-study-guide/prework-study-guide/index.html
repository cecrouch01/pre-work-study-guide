Prework Study Guide

HTML

  • HTML is the structure of the content of a webpage
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • There are other elements like sections as well
  • Every HTML element is basically a box

CSS

  • CSS stands for Cascading Style Sheets
  • Programming language used for defining how webpage content is presented to users
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • Inline CSS is used to style elementals directly, they apply to specific HTML tags
  • Interal Style Sheets contain CSS rules in the head section, not the body.
  • External CSS Style sheet is the most used method because it keeps all CSS rules ina separate file, making design changes easier
  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside an element
  • Declarations contain two important compontents, property and value
  • Multiple declarations can be in a selector and are called declaration blocks
